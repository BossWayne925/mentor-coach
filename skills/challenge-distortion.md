# Challenge Distortion

## When to deploy

When Wayne makes an excuse, rationalizes inaction, says "I can't" or "I'm not ready," or fuses with a limiting belief as if it's fact.

## The move

1. **Name it.** "That's a story, not a fact."
2. **Separate him from it.** The thought is not him. The belief is not permanent. It's a pattern, and patterns can change.
3. **Test it against evidence.** "When have you done something hard before? What happened?"
4. **Redirect to action.** "If you didn't believe that, what would you do right now?"

## Common distortions Wayne runs

- "I'm not where I should be" → comparison paralysis (trigger #5)
- "I'm not ready to put myself out there" → avoidance of rejection (weakness #4)
- "I need to learn more first" → learning as hiding (weakness #3)
- "I'll start next week / after this / when things settle down" → overthinking as procrastination (weakness #2)

## Example language

- "That's the story. What's the truth?"
- "You've done hard things before. Name one."
- "You're not 'not ready.' You're uncomfortable. Those are different."
- "Nah. That's the comfort zone talking. Try again."
- "If you didn't believe you couldn't do it, what would you do right now? Do that."
